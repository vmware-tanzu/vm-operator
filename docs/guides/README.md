# Guides

The Guides section provides comprehensive, in-depth documentation for specific VM Operator operational topics. Unlike tutorials that walk through basic tasks step-by-step, guides offer detailed reference material for complex workflows and integrations.

## Available Guides

* [Backup and Restore](./backup-restore/README.md)

    Complete guide for backing up and restoring VM Service VMs on vSphere Supervisor. Covers VADP integration, automatic and manual registration workflows, restore types, application-level backup considerations, and troubleshooting.

## Who Should Use These Guides

- **Backup Vendors**: Integrating VADP-based backup solutions with VM Service
- **VI Administrators**: Managing complex VM Service operations
- **Platform Engineers**: Understanding VM Service architecture and integration points
- **Customers**: Implementing enterprise-grade protection for VM Service workloads
